# Swift WASM Skill (Codex)

Use this skill for Swift WebAssembly setup, build, optimization, and troubleshooting in Raven projects.

## Common Invocations

- "Set up Swift WASM for this machine"
- "Build this app for wasm32-unknown-wasip1"
- "Optimize this WASM bundle size"
- "Debug this WASM runtime error"

## Scope

- toolchain setup (swiftly + Swift 6.2.x)
- WASM SDK install/verification
- debug and release builds
- production size optimization
- troubleshooting build/runtime issues

For detailed steps, see `SKILL.md`.
